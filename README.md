# Customer_Churn_Prediction
- **Objective**: Develop machine learning models to predict customer churn for improved retention strategies.
- **Tools & Frameworks**: Keras for model development, Python for data analysis, and Matplotlib/Seaborn for visualizations.
- **Data Pre-processing**: Handled missing values, scaled features, and encoded categorical variables to prepare the dataset.
- **Feature Engineering**: Extracted relevant features such as customer demographics and usage behavior to enhance model performance.
- **Model Development**: Built and trained binary classification models using algorithms like Logistic Regression, Random Forest, and Neural Networks.
- **Hyperparameter Tuning**: Optimized models to improve performance, achieving a validation accuracy of 78.62%.
- **Results Visualization**: Used data visualization techniques to present key findings, showing factors contributing to churn.
- **Insights & Strategies**: Provided actionable insights for improving customer retention, such as personalized offers and targeted customer service.
